getMetricStatistics(auth, params, interval, numOfExecutions)

Get usage statistics for a cloudwatch alarm over a specified interval. Returns an Array of Objects - each object contains metric statistics about the given execution.
Parameters:
Name Type Description

interval Number Number of ms inbetween queries (Example: for one day it would be 1000 * 60 * 60 * 24)
numOfExecutions Number How many intervals should be executed (Example: 1 week = 7 executions) Suggested: params.Period = 60, interval = 1000 * 60 * 60 * 24, numOfExecution = num of days
